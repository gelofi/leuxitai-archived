Commands List

 
The command list is updated every major update.

You can also view the commands list by doing l.help on your server.
Assume the prefix is your custom prefix.
 

Information Commands
 
Command Description Usage Aliases
help Sends the command list in the channel.
Sends a detailed help description of a specific command. 		l.help [command] h
support sends a server link to Leuxitai's server through DMs. l.support None
ping sends the API latency of the bot. l.ping p
invite sends an invite to invite Leuxitai to your server through DMs. l.invite None
weather sends the weather information at the desired location. l.weather [place] None
changelogs sends the changelog link to the website. l.changelogs chlog changelog
manual sends the instruction manual for Leuxitai. l.manual None
stats sends the status, and statistics of Leuxitai. l.stats status statistics
settings view Leuxitai's server settings. l.settings None
